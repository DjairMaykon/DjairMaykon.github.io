Olá.
Meu nome é Djair.

Eu sou um desenvolvedor apaixonado pela arte de resolver problemas e inovar com código. Acredito na recompensa pelo esforço e na importancia de cada passo em direção ao seu objetivo.

Tenho formação de tecnico em informatica pelo IFBA - Campus Jequié onde me tive a oportunidade de participar de competições de programação, robotica e aprendi a base de desenvolvimento de aplicações comerciais: WEB com HTML, CSS, Javascript e PHP; Desktop com Java;

Tenho experiencia profissional de 1 ano com desenvolvimento WEB fullstack, com PHP no backend e HTML, CSS e Javascript no frontend. Também trabalhei por 6 meses com uma aplicação desktop utilizando C# e um aplicação mobile com Java. Atualmente sou desenvolvedor fullstack na SGA Sistemas onde atuo com Java e Spring no backend e Vue.js no frontend. Além de criar e manter ferramentas de etl com python e trabalhar diretamente com o ambiente de cloud da google GCP.

Curso Ciência da Computação na UFBA desde de 2019 e participo de projeto de pesquisa no campo de datascience para saúde e para monitoramento de gastos governamentais.
